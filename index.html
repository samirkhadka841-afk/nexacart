How to Run NexaCart

NexaCart is structured primarily as a high-fidelity native Google Android APK application. Since this repository is hosted live on GitHub and integrated with Google AI Studio, follow the standard release steps below to load it on any emulator or mobile phone.

DEPLOYMENT COMPLIANCE WARNING

Do not deploy this Gradle Android project to Vercel as a core runtime. Vercel specializes exclusively in static/serverless web environments, whereas NexaCart is a native mobile APK. By hosting this index.html landing page, your Vercel URL works correctly as a global landing station without 404 blockages.

INSTALLATION STEPS RESOURCES ACTIVE

1

Build the Signed Application Package (.APK)

Within the Google AI Studio builder interface, open the developer control panel on the right side. Select Export App or trigger the direct Gradle compile commands to assemble the release APK asset target safely.

2

Sideload or Streaming Interactive Emulator

Simulate directly inside the web browser via the Streaming Android Emulator canvas, or move the compiled app-debug.apk to your Android smartphone storage, and execute the package installer.

3

Input API Security Credentials

Access the **Secrets panel** inside the AI Studio UI to load your custom Gemini keys safely. These values populate automatically via class wrappers inside the native build architecture without hardcoding variables.
